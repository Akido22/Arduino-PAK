# Automatic Meteo Station

Тестовая погодная станция собраной на Arduino. Станция определяет 
 - Температуру
 - Влажность
 - Давление
 - Освещенность
 - Дождь
 - Анализатор воздуха
 
### Оборудование и датчики

- Arduino Nano
- DHT11
- BMP280
- YL-83
- LM393
- MQ2
